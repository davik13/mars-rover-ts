# mars-rover-ts

Install dependencies
```
npm install
```

Build
```
npm run build 
```

Start the server
```
npm start
```

Run the tests
```
npm run test
```

## How to use
Go to http://localhost:3000/ and add the first letter of the command you want to execute.

```
A: Avancer
G: Gauche
D: Droite
R: Reculer
```

Example input:
```
http://localhost:3000/A
```
